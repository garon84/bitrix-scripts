# bitrix-scripts

Коллекция PHP/Python скриптов для автоматизации задач в Bitrix24 через REST API.

## Скрипты

### `check_duplicate_deals.php`
Вебхук-обработчик для проверки дублей сделок в воронке.

При срабатывании получает данные текущей сделки, ищет в той же воронке активные сделки с тем же контактом. Если дубль найден — добавляет суффикс `ДУБЛЬ!` к названию сделки и оставляет комментарий в таймлайне со списком дублирующих сделок.

Включает debug-логирование всех запросов и ответов в файл `webhook-debug.log`.

**Параметры для настройки:**
- `$dealId` — ID сделки (подставляется через шаблон `{{ID}}` в автоматизации Bitrix24)
- `$webhookUrl` — URL вебхука с правами `crm`

---

### `deal_activities.php`
Вебхук-обработчик события `ONCRMACTIVITYADD`.

Получает ID созданной активности, запрашивает её детали и удаляет, если это активность типа `IMOPENLINES_SESSION` (чат открытой линии) с нежелательным `PROVIDER_TYPE_ID`. Логирует все операции в `webhook_debug.log`.

**Параметры для настройки:**
- `$webhook_url_get` — URL вебхука с методом `crm.activity.get`
- `$webhook_url_delete` — URL вебхука с методом `crm.activity.delete`

---

### `webhook_leads.php`
HTTP-эндпоинт для приёма заявок с внешних форм (сайт, Instagram и др.).

Принимает POST-запрос с JSON-телом (`name`, `phone`, кастомные поля). Ищет контакт по телефону — если не найден, создаёт. Затем создаёт лид в Bitrix24 с привязкой к контакту и UTM-параметрами источника.

**Параметры для настройки:**
- `$webhook` — URL вебхука с правами `crm`
- Поля лида (`UF_CRM_*`) — кастомные поля, специфичные для портала

---

### `update_vat.py`
Python-скрипт для массового обновления ставки НДС у всех товаров каталога.

Выгружает все товары через `crm.product.list` с пагинацией, затем обновляет поля `VAT_ID` и `VAT_INCLUDED` пакетами по 50 штук через `batch`. Выводит прогресс и итоговую статистику.

**Параметры для настройки:**
- `WEBHOOK` — URL вебхука с правами `crm`
- `VAT_ID` — ID ставки НДС в Bitrix24
- `VAT_INCLUDED` — `"Y"` / `"N"` (включён ли НДС в цену)

## Общие требования

- PHP 7.4+ (для PHP-скриптов)
- Python 3.8+ и пакет `requests` (для `update_vat.py`)
- Вебхук Bitrix24 с нужными правами CRM
